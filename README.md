# TodoRails

A full-featured task management web application built with Spring Boot 3, Thymeleaf, and MySQL. Keep your goals on track with priority-based dashboards, deadline tracking, and secure user authentication.

---

## Features

- **User Registration & Authentication** — Register with username, email, password, qualification, and hobbies. Login via username or email with Spring Security and BCrypt password hashing.
- **Task Management** — Create, edit, view, and delete tasks with title, description, priority (High / Medium / Low), due date, and type (Official / Family / Personal).
- **Priority Dashboard** — Today's tasks grouped into High, Medium, and Low priority columns for at-a-glance focus.
- **Master Task List** — All tasks sorted by due date with alternating row styling, showing title, description, and dates.
- **Mark as Done** — Complete tasks with a single click. Completion date is automatically recorded.
- **Task Status** — Visual indicators: green for completed, red for pending.
- **Change Password** — Authenticated users can update their password securely.
- **Client-Side Validation** — Real-time form validation with user-friendly error messages.
- **Responsive Design** — Mobile-friendly layout with hamburger sidebar navigation.

## Getting Started

### Prerequisites

- **Java 21** or later
- **Maven 3.8+**
- **MySQL 8.0+**

### Setup

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd TodoRails
   ```

2. **Create the database**

   ```sql
   CREATE DATABASE todorails;
   ```

3. **Configure database credentials**

   Edit `src/main/resources/application.properties`:

   ```properties
   spring.datasource.username=your_username
   spring.datasource.password=your_password
   ```

4. **Run the application**

   ```bash
   ./mvnw spring-boot:run
   ```

   The application starts at `http://localhost:8080`.

5. **Access the app**

   - Landing page: `http://localhost:8080/`
   - Register: `http://localhost:8080/register`
   - Login: `http://localhost:8080/login`

## Usage

### Public Pages

| Route | Page | Description |
|---|---|---|
| `/` | Index | Landing page with features overview |
| `/register` | Register | Create a new account |
| `/login` | Login | Sign in with username or email |
| `/terms` | Terms | Terms and conditions |

### Authenticated Pages

| Route | Page | Description |
|---|---|---|
| `/dashboard` | Dashboard | Today's tasks by priority + master list |
| `/addtask` | Add Task | Create a new task |
| `/edittask` | Edit Task | Modify an existing task |
| `/viewtask` | View Task | Full task details |
| `/changepasswd` | Change Password | Update account password |

### API Endpoints

| Method | Route | Action |
|---|---|---|
| `POST` | `/register` | Register a new user |
| `POST` | `/addtask` | Save a new task |
| `POST` | `/updatetask` | Update an existing task |
| `POST` | `/task/markDone` | Mark a task as completed |
| `POST` | `/task/viewtask` | View task details |
| `POST` | `/task/edittask` | Load task edit form |
| `POST` | `/task/delete` | Delete a task |
| `POST` | `/changepasswd` | Change user password |

## Database

The schema is auto-generated by Hibernate (`ddl-auto=create` in development).

### Tables

- **`user`** — Stores registered users with credentials, qualification, and hobbies.
- **`user_hobbies`** — Join table for user hobby elements.
- **`task`** — Stores tasks linked to users via foreign key.

## Security

- Passwords are hashed with **BCrypt**.
- Session-based authentication via **Spring Security** form login.
- Static resources (CSS, JS, images) are publicly accessible.
- All other routes require authentication.
- CSRF protection is disabled for development convenience; **enable in production**.
